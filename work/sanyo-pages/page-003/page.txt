7, OPERATION
      After entering the program           in the LOAD     mode, the mode           switch should     be
      changed     to the RUN     mode.

      Referring to the NOTES         column,     if any data are needed         to store into the data
      memories,     data storing operation      should be completed just after the mode                    switch
      is changed    to the RUN      mode.
      Then   program     is executed by operating the items in order.

      in this column.
